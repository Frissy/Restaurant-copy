# Restaurant landing page / Pré-formation

Ce petit projet était un exercice autodidacte de reproduction de la maquette d'un site web d'un restaurant. Il date de 2 ans avant le début de ma formation de Développeur Web et Web Mobile, je l'ai laissé comme tel de manière volontaire, afin de pouvoir suivre l'évolution de la qualité de mon code au fur et à mesure que je gagne en expérience.

Il est hébergé à l'adresse suivante : https://frissy.github.io/restaurant-copy/
